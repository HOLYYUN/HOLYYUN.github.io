安于做梦

作者: 十三 时间: 五月 5, 2019  14:15:41 分类: --随想

刚开始被越来越多的梦侵占睡眠的时候，并没有把这件事放在心上，只是感叹一下那些清晰记得的梦为什么都是些已经离开的人
到后来读大学，毕业工作，属于个人的时间和空间不断被压缩时，惶恐的发现可能做梦是自己发泄这些多余想象力的唯一异度时间了

于是开始享受起做梦这件事，不仅是因为那种似是而非的真实感，也是为了那些可能不可能再见面的人
一叶扁舟摇摇晃晃终归是要驶向自己的终点。
